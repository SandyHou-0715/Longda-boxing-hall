# 龙达搏击馆 - 会员管理系统

> 微信小程序版 · 搏击馆会员管理系统 · 暗黑酷炫风

## 📖 项目介绍

龙达搏击馆会员管理系统是一款基于**微信小程序原生开发**（WXML + WXSS + JS + JSON）的会员管理应用，专为搏击/武术馆设计。集成自由搏击、拳击、散打、跆拳道等多项目管理，使用 Mock 数据模拟后端，无需真实服务器即可完整体验所有功能。

## ✨ 功能特性

| 功能模块 | 说明 |
|---------|------|
| 🔐 用户登录 | 手机号 + 密码登录，支持登录状态持久化 |
| 🏟 选择校区 | 登录后先选择校区（朝阳/海淀/丰台/通州），支持在个人中心切换 |
| 🏠 首页 | 轮播横幅、公告栏、当前校区信息、数据概览、近期课程、教练预览、**会员卡购买模块** |
| 💳 会员卡购买 | 首页内嵌次卡/月卡/年卡横向轮播，点击进入详情/购买 |
| 📅 我的课程表 | 查看已预约课程，支持状态筛选，可取消预约 |
| 📋 约课 | 浏览未来14天可预约课程，**新增教练筛选**（横向头像标签），一键预约/取消 |
| 🛒 微商城 | 搏击/跆拳道装备商城，顶部搜索、分类导航、网格商品、加入购物车 |
| 🛍 购物车 | 商品数量调整、小计、一键结算（Mock） |
| 📦 商品详情 | 商品图片、价格、描述、数量选择、加购/立即购买 |
| ✅ 签到打卡 | 从Tab移至个人中心菜单，月度日历、连续签到奖励 |
| ⭐ 积分中心 | 积分余额展示、变动记录、积分规则说明 |
| 👊 教练团队 | 教练列表及详情（专项、荣誉、评分、课程） |
| 📚 上课记录 | 已完成课程记录，按月分组显示 |
| 👤 个人中心 | 用户资料、数据统计、功能菜单（含切换校区、签到打卡）、退出登录 |
| 🏟 关于我们 | 场馆介绍、联系信息、发展历程、场馆环境 |
| ⚙️ 设置 | 通知开关、音效开关、清除缓存、版本信息 |

## 🗂 底部 TabBar

|-----|------|------|
| 🏠 首页 | pages/index/index | 主页 |
| 📅 约课 | pages/booking/booking | 课程预约（含教练筛选） |
| 🛒 商城 | pages/shop/shop | 微商城（装备购买） |
| 👤 我的 | pages/profile/profile | 个人中心 |

## 🛠 技术栈

- **微信小程序原生开发**（WXML + WXSS + JS + JSON）
- 无第三方框架（不使用 Taro、uni-app 等）
- 使用 `wx.setStorageSync` 实现数据持久化
- 全部数据使用 Mock 数据，无需真实后端

## 🚀 如何运行

### 使用微信开发者工具打开

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，点击「+」新建项目
3. 选择「导入项目」，选择本项目根目录（`Longda-boxing-hall/`）
4. AppID 可填写测试号（点击"测试号"选项）或填写 `wx1234567890abcdef`
5. 点击「导入」，即可在模拟器中运行

> **注意**：项目根目录直接就是微信小程序标准结构，包含 `app.js`、`app.json` 等入口文件，可直接用微信开发者工具打开。

## 📁 项目结构

```
Longda-boxing-hall/
├── app.js                  # 小程序入口，全局状态初始化

│   ├── mock.js             # Mock 数据（用户、教练、课程、校区、会员卡、商品等）

│
├── images/
│   └── tabbar/             # 底部导航栏图标（灰色+彩色各4个PNG）
│
└── pages/
    ├── login/              # 登录页（手机号+密码登录）
    ├── campus/             # 选择校区（新增）
    ├── index/              # 首页（轮播图、公告、校区信息、会员卡购买）
    ├── booking/            # 约课（含教练筛选功能）
    ├── shop/               # 微商城（新增，替代签到Tab）
    ├── shopDetail/         # 商品详情（新增）
    ├── cart/               # 购物车（新增）
    ├── cardDetail/         # 会员卡购买详情（新增）
    ├── checkin/            # 签到打卡（从Tab移至个人中心入口）
    ├── schedule/           # 我的课程表
    ├── points/             # 积分中心
    ├── coaches/            # 教练团队列表
    ├── coachDetail/        # 教练详情（简介、荣誉、课程）
    ├── records/            # 上课记录（按月分组）
    ├── membercard/         # 会员卡（卡片式展示、课时进度）
    ├── profile/            # 个人中心（含切换校区、签到打卡菜单）
    ├── about/              # 关于我们（场馆介绍、联系方式）
    └── settings/           # 设置（通知、音效、清缓存）
```
