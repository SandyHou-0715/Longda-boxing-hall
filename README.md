# 龙达搏击馆 - 会员管理系统

> 微信小程序版 · 搏击馆会员管理系统

## 📖 项目介绍

龙达搏击馆会员管理系统是一款基于**微信小程序原生开发**（WXML + WXSS + JS + JSON）的会员管理应用，专为搏击/武术馆设计。集成自由搏击、拳击、散打、跆拳道等多项目管理，使用 Mock 数据模拟后端，无需真实服务器即可完整体验所有功能。

## ✨ 功能特性

| 功能模块 | 说明 |
|---------|------|
| 🔐 用户登录 | 手机号 + 密码登录，支持登录状态持久化 |
| 🏠 首页仪表盘 | 轮播横幅、公告栏、数据概览、快捷入口、近期课程、教练预览 |
| 📅 我的课程表 | 查看已预约课程，支持状态筛选，可取消预约 |
| 📋 约课 | 浏览未来14天可预约课程，一键预约/取消 |
| ✅ 签到打卡 | 每日签到获积分，查看月度签到日历，连续签到奖励 |
| ⭐ 积分中心 | 积分余额展示、变动记录、积分规则说明 |
| 👊 教练团队 | 教练列表及详情（专项、荣誉、评分、课程） |
| 📚 上课记录 | 已完成课程记录，按月分组显示 |
| 💳 会员卡 | 卡片式会员信息展示，课时进度条，权益说明 |
| 👤 个人中心 | 用户资料、数据统计、功能菜单、退出登录 |
| 🏟 关于我们 | 场馆介绍、联系信息、发展历程、场馆环境 |
| ⚙️ 设置 | 通知开关、音效开关、清除缓存、版本信息 |

## 🛠 技术栈

- **微信小程序原生开发**（WXML + WXSS + JS + JSON）
- 无第三方框架（不使用 Taro、uni-app 等）
- 使用 `wx.setStorageSync` 实现数据持久化
- 全部数据使用 Mock 数据，无需真实后端

## 🚀 如何运行

### 使用微信开发者工具打开

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，点击「+」新建项目
3. 选择「导入项目」，选择本项目根目录（`Longda-boxing-hall/`）
4. AppID 可填写测试号（点击"测试号"选项）或填写 `wx1234567890abcdef`
5. 点击「导入」，即可在模拟器中运行

> **注意**：项目根目录直接就是微信小程序标准结构，包含 `app.js`、`app.json` 等入口文件，可直接用微信开发者工具打开。

## 📁 项目结构

```
Longda-boxing-hall/
├── app.js                  # 小程序入口，全局状态初始化
├── app.json                # 全局配置（页面路由、tabBar、窗口样式）
├── app.wxss                # 全局样式（配色主题、公共组件样式）
├── project.config.json     # 微信开发者工具项目配置
├── sitemap.json            # 微信搜索站点地图配置
│
├── utils/
│   ├── mock.js             # Mock 数据（用户、教练、课程、签到记录等）
│   └── util.js             # 工具函数（日期格式化、进度计算等）
│
├── images/
│   └── tabbar/             # 底部导航栏图标（灰色+彩色各4个PNG）
│
└── pages/
    ├── login/              # 登录页（手机号+密码登录）
    ├── index/              # 首页（轮播图、公告、快捷入口、教练预览）
    ├── booking/            # 约课（浏览并预约未来课程）
    ├── checkin/            # 签到打卡（日历、连续签到奖励）
    ├── schedule/           # 我的课程表
    ├── points/             # 积分中心
    ├── coaches/            # 教练团队列表
    ├── coachDetail/        # 教练详情（简介、荣誉、课程）
    ├── records/            # 上课记录（按月分组）
    ├── membercard/         # 会员卡（卡片式展示、课时进度）
    ├── profile/            # 个人中心
    ├── about/              # 关于我们（场馆介绍、联系方式）
    └── settings/           # 设置（通知、音效、清缓存）
```
